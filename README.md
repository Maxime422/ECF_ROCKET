## 📦 Installation

1. **Cloner le dépôt :**
    ```bash
    git clone https://github.com/Maxime422/ECF_ROCKET.git
    ```
2. **Se rendre dans le dossier du projet :**
    ```bash
    cd ECF_ROCKET
    ```
3. **Installer les dépendances :**
    ```bash
    npm install
    ```
4. **Lancer le projet :**
    ```bash
    npm start
    ```
